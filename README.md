# 👁️ AI-ассистент для слабовидящих

Интеллектуальная система помощи людям с нарушениями зрения на основе компьютерного зрения, OCR и синтеза речи. Система анализирует окружающую среду через камеру смартфона и озвучивает информацию пользователю в режиме реального времени.

---

## 🎯 Ключевые возможности

| Модуль | Описание |
|--------|----------|
| 🔍 **Детекция объектов** | Обнаружение препятствий, людей, транспорта и других объектов в реальном времени |
| 📏 **Depth Estimation** | Оценка расстояния до объектов: «Лестница через два метра», «Препятствие справа» |
| 📝 **OCR** | Распознавание текста на вывесках, ценниках, упаковках |
| 💵 **Распознавание купюр** | Гибридный алгоритм: детекция → OCR номинала → классификация-подтверждение |
| 💊 **Поиск лекарств** | Нечёткий поиск по базе препаратов — находит совпадения даже при частично нечитаемой упаковке |
| 🔊 **TTS с приоритизацией** | Голосовые подсказки по приоритетам, без перегрузки пользователя |

## 🛠️ Стек технологий

### Computer Vision & Deep Learning
- **[YOLOv8](https://github.com/ultralytics/ultralytics)** — детекция объектов в реальном времени (backbone + neck + head)
- **[PyTorch](https://pytorch.org/)** — фреймворк для обучения и вывода моделей
- **[OpenCV](https://opencv.org/)** — препроцессинг изображений, работа с кадрами

### OCR
- **[EasyOCR](https://github.com/JaidedAI/EasyOCR)** / **[PaddleOCR](https://github.com/PaddlePaddle/PaddleOCR)** — распознавание текста
- Постпроцессинг: нормализация регистра, исправление опечаток, сверка со словарём

### Depth Estimation
- **[MiDaS](https://github.com/isl-org/MiDaS)** — монокулярная оценка глубины
- **ZoeDepth** — метрическая оценка расстояний
- **DPT (Dense Prediction Transformer)**

### Распознавание купюр
- Гибридный pipeline: YOLO-детекция → OCR → CNN-классификатор-подтверждение
- Обучение на датасете российских рублей (Позднее, обучение на других валютах)

### Нечёткий поиск по лекарствам
- **[RapidFuzz](https://github.com/rapidfuzz/RapidFuzz)** — быстрый нечёткий поиск по базе препаратов
- База данных лекарственных средств (ГРЛС / локальная SQLite-база)

### Оптимизация под мобильные устройства
- **ONNX Runtime** — кроссплатформенный вывод моделей
- **OpenVINO** — ускорение на Intel-чипах
- Квантование (float32 → int8), прунинг, дистилляция знаний

### Синтез речи (TTS)
- **[pyttsx3](https://github.com/nateshmbhat/pyttsx3)** — полностью офлайн TTS
- **[edge-tts](https://github.com/rany2/edge-tts)** — качественный TTS при наличии сети

### Backend / API
- **[FastAPI](https://fastapi.tiangolo.com/)** — REST API для серверной обработки
- **SQLite** — локальная база лекарств и настроек

### Мобильное приложение
- **Flutter/Dart** (Позднее, возможно, отдельно под iOS, отдельно под Android)

## ⚙️ Требования к производительности

- Частота обработки: **≥ 15–30 FPS** в режиме реального времени
- Задержка голосового оповещения: **< 500 мс**
- Работа в условиях: плохое освещение, motion blur, шум изображения
- Платформа: Android-смартфон среднего класса (без GPU-сервера)
